# Number Guess Game

A simple **C# console game** where the player guesses a randomly generated number between 1 and 100. Designed as a fun beginner-friendly project while practicing C# fundamentals.

- Random number generation
- User input validation
- Keeps track of attempts
- Friendly console interface
- Replayable game
